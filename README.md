# ahs-hacks
GrowGuide is a modern, gardening assistant that helps anyone grow healthy plants with confidence. Using location and weather data, it delivers tailored care guides, plant recommendations, progress tracking, and reminders. Explore a full plant library, manage your garden, and discover recipes for what you grow.
